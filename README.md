#### 已实现功能点:
*   基于TestNg+PageObject的UI功能测试
*   基于ExtentReport的测试报告
*   应用性能指标收集和图表绘制(CPU、FPS、流量、启动速度等)
*   Excel和Yaml的数据驱动
*   结合Jenkins结合实现持续构建
*   Crash截图和重试
*   简单的图片比对
